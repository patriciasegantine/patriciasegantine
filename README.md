## About Me 🚀

Hello! I'm a dynamic and passionate Front-end Developer with over 5 years of experience, specialising in creating innovative solutions for B2B, B2C, and SaaS platforms. I love contributing to digital transformations and developing customised software applications.

I'm experienced in front-end technologies like JavaScript (ES6), React.js, TypeScript, Redux, Jest, RESTful API, Git, HTML, CSS, and Styled Components. I'm also skilled in Azure DevOps, CI/CD, and SCRUM Agile methodologies.

Achievements 🏆

I've helped companies enhance their digital presence and achieve their goals by:

Leading critical feature development for B2B SaaS platforms, cutting down project development time.
Using React, TypeScript, Redux, and Context API for state management and modular component design, speeding up feature development.
Improving development processes by working closely with UX teams and adopting Ant Design and Styled Components, boosting user satisfaction.
Creating and integrating front-end microservices, allowing for faster delivery of new features.
Implementing CI/CD on Microsoft Azure DevOps, reducing deployment times and improving team collaboration.
